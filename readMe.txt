Client Folder

1. Open the terminal.
2. Type "npm install" to get the node modules.
3. To run the client, type "npm start"

Server Folder

1. Open the terminal.
2. Type "npm install" to get the node modules.
3. Go to config/key.js.
4. Kindly put your own MONGO URI to connect to the database.
5. Save it.
6. To run the server, type "npm start"
